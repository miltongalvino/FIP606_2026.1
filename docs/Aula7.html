Aula7: Modelos Mistos e Parcelas Subdivididas (Split-Plot)

Introdução: Parcelas Subdivididas (Split-Plot)

Nesta aula, trabalharemos com um delineamento em parcelas subdivididas (split-plot), comum quando temos dois fatores mas restrições de aleatorização no campo. O fator principal é aplicado à parcela inteira (main plot) e o fator secundário à subparcela (subplot).

Primeiro, carregamos e inspecionamos os dados graficamente para entender a distribuição do índice/produtividade para cada híbrido sob diferentes métodos.

Abordagem 1: ANOVA Clássica com Termo de Erro

Na abordagem clássica via aov, precisamos explicitar o erro da parcela principal, já que o modelo de parcelas subdivididas possui dois níveis de erro experimental.

O termo Error(block/hybrid) define que hybrid está aninhado dentro de block, especificando corretamente o erro da parcela principal.

Abordagem 2: Modelos Mistos (lme4 e lmerTest)

A abordagem moderna e recomendada para delinamentos complexos (como parcelas subdivididas ou medidas repetidas) é o uso de Modelos Mistos. O bloco e o erro da parcela principal são tratados como efeitos aleatórios.

Testaremos as premissas de ambos os modelos e utilizaremos o modelo misto (lmer) para a inferência final.

Resultados do Teste de Médias

Quando a interação entre dois fatores (Híbrido x Método) é significativa, é praxe na literatura acadêmica construir uma tabela dupla ou um gráfico de barras contendo as letras maiúsculas e minúsculas correspondentes ao agrupamento do teste de médias.

A tabela abaixo foi formatada dinamicamente utilizando kableExtra, que fornece uma estilização html profissional.

library(knitr)
library(kableExtra)

Gráfico de Barras com Agrupamento Estatístico

Outra forma elegante de apresentar a interação significativa é usando gráficos de barras emparelhados (position_dodge), preenchidos pelo método, contendo barras de erro baseadas no Erro Padrão (SE) e os rótulos de agrupamento (geom_text).
